Research Vision

My work sits at the intersection of behavioral machine learning and socio-technical system resilience. I focus on:

  • Detecting social engineering through behavioral anomaly modeling
  • Predicting system failures (e.g., programmer attrition, food spoilage, road accidents) using interpretable ML
  • Developing continual learning frameworks for dynamic environments

I aim to build AI systems that are not only accurate but explainable, fair, and grounded in real-world behavioral data.

Research Projects

Programmer Attrition Prediction (BIM 2025)

Socio-economic ML framework using behavioral + demographic features to predict developer retention. Achieved high interpretability via SHAP.

Methods: XGBoost, SHAP, cross-validation

Feature Fusion for Attrition Modeling (BIM 2025)

Complementary approach using fused behavioral traces and organizational metadata to improve prediction stability.

Methods: Ensemble modeling, feature selection

SafeRoads — Geospatial Risk Prediction

ML system to identify high-risk urban road segments using traffic, weather, and accident history. Supports risk-aware navigation.

Methods: Geospatial ML, ensemble models

MeteorShield — NASA Space Apps 2025

Planetary defense platform using NASA NEO data with real-time 3D visualization. Champion, Barisal Division.

Tech: NASA APIs, Three.js, data storytelling
